# Currency Formatter 💱

A simple and intuitive web application for formatting currencies with precision and style! 💰✨

Easily convert and display currency values in various formats, making financial data more readable and professional. Perfect for developers, accountants, and anyone working with monetary values. 📊💼

🔗 **Live Demo**: [https://iam269.github.io/Currency-Formatter/](https://iam269.github.io/Currency-Formatter/)

## Features 🌟
- **Multi-Currency Formatting**: Instantly format amounts into Indian Rupees (INR), US Dollars (USD), and Japanese Yen (JPY) 🇮🇳🇺🇸🇯🇵
- **Real-Time Conversion**: Enter any numeric value and see formatted results immediately 🔄
- **Responsive Design**: Works seamlessly on desktop and mobile devices 📱💻
- **Clean UI**: Modern interface with gradient backgrounds and smooth shadows 🎨
- **Intl API Integration**: Uses JavaScript's Internationalization API for accurate locale-specific formatting 🌐

## How to Use 📝
1. Enter a numeric amount in the input field
2. Click the "Format" button
3. View the formatted currencies below

## Technologies Used 🛠️
- **HTML5**: Semantic markup for structure
- **CSS3**: Custom styling with gradients and responsive design
- **JavaScript**: ES6+ with Intl.NumberFormat for currency formatting
- **Google Fonts**: Roboto Mono for a clean, monospace look

Built with HTML, CSS, and JavaScript. 🚀
